## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/amineNouabi/play-position.git
```

2. Go into project

```bash
cd play-position/landing-page
```

3. Install dependencies

```bash
pnpm install
```

4. Run project

```bash
pnpm run dev
```
